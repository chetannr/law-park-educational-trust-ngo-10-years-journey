# unDraw Illustrations

This folder should contain unDraw illustrations downloaded from [unDraw.co](https://undraw.co).

## How to Add Illustrations

1. Visit [unDraw.co](https://undraw.co)
2. Search for relevant illustrations (e.g., "education", "children", "community", "donation", "team")
3. Set the primary color to `#16a34a` (your brand green color) using the color picker
4. Download the SVG files
5. Rename and place them in this folder with these names:

### Recommended Illustrations:

- `educator.svg` - For education/teaching themes
- `children.svg` - For children/students
- `community.svg` - For community engagement
- `growth.svg` - For impact/growth
- `team.svg` - For volunteers/team
- `donation.svg` - For donations/support
- `graduation.svg` - For scholarships/achievements
- `books.svg` - For libraries/reading

## Alternative: Use CDN

If you prefer not to host illustrations locally, you can modify the `Illustration.tsx` component to use a CDN service that hosts unDraw illustrations.

## Color Reference

- Primary color for illustrations: `#16a34a` (green)
- White for illustrations on dark backgrounds: `#ffffff`
